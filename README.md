# Full Page Nav 🌐

A sleek and responsive full-page navigation system designed for modern web applications. This project showcases smooth scrolling, interactive elements, and a clean design that enhances user experience. 🚀✨

## Features
- 📱 Responsive design for all devices
- 🎨 Customizable styles and animations
- ⚡ Lightweight and fast loading
- 🔗 Easy integration with existing projects

## Live Demo
Check out the live demo here: [https://itsiamdev.github.io/Full-Page-Nav/](https://itsiamdev.github.io/Full-Page-Nav/) 🌟

## Getting Started
1. Clone the repository
2. Open `index.html` in your browser
3. Customize the styles in `style.css` as needed
